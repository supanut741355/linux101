Redirection คือการนำ standardXXX ไปทำลงไฟล์

(แวะ) standXXX:
- standard input
- standard output
- standard error 


การ Redirection มี 2 ท่า
1. ใช้ > , >> สำหรับ stdout


2. ใช้ 2> , 2>> สำหรับ stderr

3. ใช้ | tee 


(แวะ) pipe ( | ) สามารถใช้ chain command ได้

 "เอา output ของ command_1 มาเป็น input ของ command_2"
